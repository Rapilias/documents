◇スイーパー(全5種)◇

ゲーム「ブルーアーカイブ」に登場する敵キャラクターであるスイーパーの非公式3Dモデルデータです。
付属している.unitypackageではUniversalRenderPipelineのマテリアルが適応されたモデルが同梱されています。
利用に際しての技術的な質問への回答は出来かねますので予めご了承ください。

◇利用規約◇

・「ブルーアーカイブ」二次創作・ゲーム実況配信及び動画投稿に関するガイドライン( https://bluearchive.jp/news/newsJump/116 )を遵守して下さい。
・本モデルの商用利用を禁止します。
・本モデルの改変/未改変を問わず、二次配布は禁止します。
  ・ただし、ゲーム/アプリ内での利用に限り、改変/未改変問わず利用が可能です。
    ・ゲーム/アプリを配布する場合は、アセットになんらかの加工を行って直接取り出せないようにしてください。(.fbxや.pngを直接公開しないでください。)
・当利用規約は予告なく改定する場合があります。利用規約は最新のものが適用されます。
・本モデルの使用により発生した損害、損失、不利益に対して当方は一切責任を負いません。全て自己責任の上で使用をお願いします。
・本モデルの利用者はモデルデータの利用をもって本利用規約に同意したものとみなします。
